Para poder ejecutar el archivo de python sin problemas se requieren instalar ciertos paquetes, para ello
abrir la consola en la ruta base y ejecutar la siguiente linea de codigo

	pip install -r requirements.txt

Ejecutar el archivo "extract_features.py", en la consola se mostran los resultados, las imagenes utilizadas se encuentran en la carpeta BDN, separadas en test y train.
